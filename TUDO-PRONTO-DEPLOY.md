## 🐛 Troubleshooting Rápido

### Build Falha
```bash
rm -rf node_modules dist
npm install
npm run build
```

### Botão Instalar Não Aparece
- ✅ Já está instalado? (Normal)
- ✅ Está em HTTPS? (Vercel faz automático)
- ✅ Service Worker ativo? (F12 → Application)

### Site Não Carrega
- Aguarde 5 minutos (DNS)
- Limpe cache (Ctrl+Shift+R)
- Tente modo anônimo

### PWA Não Funciona Offline
- Verifique Service Worker (F12 → Application)
- Limpe cache
- Reinstale o app
