## 🚀 Passo a Passo

### 1️⃣ Criar Conta Vercel (2 minutos)

1. Acesse: **https://vercel.com**
2. Clique em **"Sign Up"**
3. Escolha **"Continue with GitHub"**
4. Autorize o Vercel

✅ **Pronto!** Conta criada.

---

### 2️⃣ Fazer Deploy (3 minutos)

1. No Vercel Dashboard, clique em **"New Project"**
2. Clique em **"Import Git Repository"**
3. Selecione o repositório **planocerto**
4. Configure:
   ```
   Framework Preset: Vite
   Build Command: npm run build
   Output Directory: dist/client
   Install Command: npm install
   ```
5. Clique em **"Deploy"**
6. ☕ Aguarde 2-3 minutos

✅ **Pronto!** Site no ar.

---

### 3️⃣ Testar (5 minutos)

1. **Acesse a URL** fornecida (ex: `planocerto.vercel.app`)
2. **Navegue** pelo site
3. **Teste o botão** "Instalar App" (canto inferior direito)
4. **Instale no celular**:
   - Android: Clique no botão "Instalar App"
   - iOS: Botão compartilhar → "Adicionar à Tela de Início"
5. **Teste offline**:
   - Ative modo avião
   - Abra o app instalado
   - Deve funcionar!

✅ **Pronto!** Tudo funcionando.

## 🆘 Problemas?

### Build Falhou
```bash
# Teste local primeiro
npm run build

# Se funcionar local, tente novamente no Vercel
```

### Botão "Instalar App" Não Aparece
- ✅ Normal se já estiver instalado
- ✅ Normal no Safari iOS (use método manual)
- ✅ Precisa estar em HTTPS (Vercel já faz isso)

### Site Não Abre
- Aguarde 5 minutos (propagação DNS)
- Limpe cache do navegador (Ctrl+Shift+R)
- Tente em modo anônimo
