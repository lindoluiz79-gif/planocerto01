## 🎮 Como Usar

### 1. Rodar Localmente:
```bash
npm run dev
```

### 2. Testar Funcionalidades:

**PWA:**
- Instale como app (ícone no navegador)
- Teste offline (desative WiFi)

**Notificações:**
- Configurações → Ativar notificações
- Teste com botão de teste
- Crie alertas de preço

**Temas:**
- Configurações → Escolha um tema
- Exporte e compartilhe

**Filtros:**
- Buscar → Configure filtros → Salvar
- Filtros Salvos → Use quando quiser

**Dashboard:**
- Veja favoritos, alertas e notificações
- Gerencie tudo em um lugar

### 3. Build para Produção:
```bash
npm run build
```
