## 🎯 Comando Único para Criar o Repositório

Copie e cole este comando no PowerShell:

```powershell
.\criar-repo-github.ps1
```

**Pressione Enter e siga as instruções!**

## 📋 O Que Vai Acontecer:

1. ✅ Verifica se GitHub CLI está instalado
2. 🔐 Faz login no GitHub (se necessário)
3. 📦 Cria o repositório "PlanoCerto01"
4. 📤 Faz push de todo o código
5. 🎉 Pronto!

## 🔐 Login no GitHub

Se pedir para fazer login:

1. **Escolha:** GitHub.com
2. **Protocolo:** HTTPS
3. **Autenticar Git:** Yes
4. **Como autenticar:** Login with a web browser
5. **Copie o código** que aparecer
6. **Pressione Enter**
7. **Cole o código** no navegador
8. **Autorize**

✅ **Pronto!** Login feito!

## 🆘 Se Der Erro

### "gh: command not found"

Feche e abra o PowerShell novamente.

### "not logged in"

Execute:
```powershell
gh auth login
```

### "repository already exists"

Ótimo! O repositório já foi criado! Veja:
```powershell
gh repo view PlanoCerto01 --web
```
