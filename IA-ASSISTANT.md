## 🎯 Funcionalidades da IA

### 1. **Recomendação Inteligente de Planos**
- Faz perguntas sobre orçamento
- Pergunta sobre dependentes
- Verifica necessidade de odonto
- Recomenda os melhores planos

**Exemplo de conversa:**
```
Usuário: "Preciso de um plano"
IA: "Qual é o seu orçamento mensal?"
Usuário: "Até R$ 200"
IA: "Você precisa incluir dependentes?"
Usuário: "Não, só para mim"
IA: "Encontrei 3 planos perfeitos para você..."
```

### 2. **Explicação de Termos Técnicos**
- Carência
- Coparticipação
- ANS
- Reembolso
- E todos os termos do glossário

**Exemplo:**
```
Usuário: "O que é carência?"
IA: "Carência é o tempo de espera entre contratar o plano e usar certos serviços:
• Urgência: 24 horas
• Consultas: 30 dias
• Cirurgias: 180 dias..."
```

### 3. **Comparação de Planos**
- Compara operadoras
- Mostra diferenças de preço
- Destaca coberturas
- Avaliações

**Exemplo:**
```
Usuário: "Comparar Unimed e Hapvida"
IA: "Unimed MEI - R$ 245/mês
⭐ 4.8 | 🦷 Odonto | 🏥 Premium

Hapvida Essencial - R$ 159/mês
⭐ 4.5 | 🦷 Odonto | ❌ Premium"
```

### 4. **Informações Específicas de Planos**
- Detalhes de cada operadora
- Preços atualizados
- Coberturas
- Avaliações

**Exemplo:**
```
Usuário: "Quanto custa a Unimed?"
```

### 5. **Calculadora de Economia**

- Requisitos
