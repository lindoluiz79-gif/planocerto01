## 🆘 Se Algo Der Errado

### Build Falha no Cloudflare

**Improvável, mas se acontecer:**

1. Verifique se usou `dist/client` como output
2. Verifique se o build command é `npm run build`
3. Adicione `NODE_VERSION=18` nas variáveis de ambiente
4. Tente fazer rebuild (botão "Retry deployment")

### Site Não Carrega

1. Aguarde 5 minutos (propagação DNS)
2. Limpe cache do navegador (Ctrl+Shift+R)
3. Tente em modo anônimo
4. Verifique logs do build

### 404 em Rotas

**Não deve acontecer!** TanStack Start cuida do roteamento SSR.

Se acontecer, me avise imediatamente.
