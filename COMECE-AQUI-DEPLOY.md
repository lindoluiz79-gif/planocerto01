## ⚡ VERSÃO SUPER RÁPIDA (5 MINUTOS)

1. **Acesse**: https://dash.cloudflare.com/
2. **Login** (ou crie conta grátis)
3. **Clique**: Workers & Pages → Create application → Pages → Connect to Git
4. **Selecione**: Seu repositório `planocerto01`
5. **Configure**:
   - Build command: `npm install --legacy-peer-deps && npm run build`
   - Build output: `dist/client`
   - Environment: `NODE_VERSION=20`
6. **Deploy**: Clique em "Save and Deploy"
7. **Aguarde**: 2-3 minutos
8. **Pronto**: Seu site está no ar! 🎉
