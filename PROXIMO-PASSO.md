### Opção 2: Linha de Comando (Manual)

**Tempo: 3 minutos**

#### Passo 1: Criar Repositório no GitHub
1. Acesse: https://github.com/new
2. Nome: `PlanoCerto01`
3. Descrição: `Plataforma completa para comparação de planos de saúde`
4. Visibilidade: **Public**
5. **NÃO** marque nenhuma opção (README, .gitignore, license)
6. Clique em **"Create repository"**

#### Passo 2: Fazer Push
No terminal, execute:

```bash
# Substitua SEU-USUARIO pelo seu usuário do GitHub!
git remote add origin https://github.com/SEU-USUARIO/PlanoCerto01.git
git branch -M main
git push -u origin main
```

**Se pedir senha:** Use Personal Access Token
- Crie em: https://github.com/settings/tokens
- Marque: `repo` (Full control)
- Cole o token no lugar da senha
